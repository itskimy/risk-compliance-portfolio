# Risk & Compliance Data Analysis Portfolio

This repository showcases data analysis projects focused on transaction monitoring,
fraud risk detection, and regulatory compliance.

The projects simulate real-world scenarios commonly encountered in financial services,
fintech, and regulated environments, with an emphasis on explainable, rule-based risk analysis.

## Tools & Technologies
- SQL (MySQL)
- Python (Pandas, NumPy)
- Excel
- Power BI

## Portfolio Projects
- Transaction Monitoring & Fraud Risk Analysis
- KYC Customer Risk Profiling
- Compliance Monitoring Dashboard

## Focus Areas
- Rule-based risk detection
- Transaction monitoring
- Data quality and validation
- Risk scoring and classification
- Compliance-style reporting

> Note: All datasets used are synthetic or publicly available and are intended for demonstration purposes only.
